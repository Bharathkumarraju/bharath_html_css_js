This zip file contains code for chapters 1-12 of Beginning HTML and CSS. There is no code in Chapter 13 or in the various appendices.

With the exception of several files in Chapter 11 covering Ajax (pointed out in the text) that require a web server, all of the HTML files in this zip file should be accessible with just a web browser. Simply click on the HTML file. 

For best results with some of the newer features, use the latest versions of Google Chrome or Mozilla Firefox or Internet Explorer 10+.

Remember, these files are here for you to learn from, so look at the source code, experiment and have fun.
